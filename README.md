# NWSSU Queue Management System

A fully interactive web prototype for the Northwestern Samar State University Registrar's Office Queue Management System.

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- React Router DOM v6
- Recharts (charts)
- qrcode.react (QR codes)
- lucide-react (icons)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view the app.

## Features

- **Student Portal**: Join queues, track position, QR verification
- **Staff Portal**: Queue management, window assignment, reports
- **Role-based access**: Administrator, Registrar Staff, Window Operator
- Mock data only - no backend required
